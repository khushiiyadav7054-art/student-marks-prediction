# Student Marks Prediction

## Project Overview
This project predicts student marks based on study hours using Machine Learning.

## Algorithm Used
Linear Regression

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Dataset contains:
- Study Hours
- Marks

## Steps
1. Import libraries
2. Load dataset
3. Data visualization
4. Train model
5. Predict marks

## Result
The model predicts marks based on study hours.
